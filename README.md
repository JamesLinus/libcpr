libcprime
=========

A modern, sane, and minimalistic standard library supplement for the
venerable C programming language.

Dependencies
------------

* GCC (>= 4.x)
* Autoconf (>= 2.60)
* Automake (>= 1.11)
* Libtool (>= 2.2)

Installation
------------

    $ autoreconf --install --force
    $ ./configure
    $ make
    $ sudo make install
    $ sudo ldconfig             # on Linux
