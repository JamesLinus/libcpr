Installation on Mac OS X
========================

From a source tarball
---------------------

.. code-block:: bash

   $ ./autogen.sh
   $ ./configure
   $ make
   $ sudo make install

Using Homebrew
--------------

.. code-block:: bash

   $ brew install https://raw.github.com/bendiken/libcpr/master/etc/homebrew/libcpr.rb

.. _Homebrew: http://mxcl.github.com/homebrew/
